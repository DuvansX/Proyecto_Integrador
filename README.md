#  Proyecto Integrador Reto Santoto 30 años

## Descripción
El Proyecto Integrador es un proyecto académico desarrollado como parte del proceso formativo universitario, cuyo propósito es el diseño y desarrollo de una base de datos unificada para la gestión de trabajadores. Este proyecto vincula a estudiantes de primer y segundo semestre de los programas de Sistemas e Datos-Inteligencia Artificial. 

En el contexto del proyecto, la Gobernación de Boyacá, a través de la Dirección General de Talento Humano, plantea como reto la creación de una base de datos que permita consolidar la información actualmente dispersa en múltiples fuentes, con el objetivo de facilitar su consulta en tiempo real y mejorar la toma de decisiones administrativas. 

El objetivo principal del proyecto es integrar conocimientos de programación en Java para la construcción de una solución que permita organizar, procesar y gestionar información de manera estructurada, aplicando buenas prácticas de desarrollo de software. 

Como entrega final, el proyecto contempla la implementación de la base de datos junto con un dashboard de analítica preliminar.
